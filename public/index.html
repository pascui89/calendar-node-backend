Authentication

Description

Log in a user.

Request Body

JSON object containing the user's email and password.

Response

JSON object containing the user's ID, name, and a token.

Description

Create a new user.

Request Body

JSON object containing the user's email, password, and other details.

Response

JSON object containing the created user's ID, name, and a token.

Description

Renew a user's token.

Response

JSON object containing a new token.

Events

Description

Retrieve all events.

Response

JSON object containing the list of events and a token.

Description

Create a new event.

Request Body

JSON object containing the event details.

Response

JSON object containing the created event and a token.

Description

Update an existing event.

Request Body

JSON object containing the updated event details.

Response

JSON object containing the updated event and a token.

Description

Delete an existing event.

Response

JSON object confirming the deletion and a token.
